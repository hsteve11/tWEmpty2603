### 📝 Instant Pad / Shareable URL Editors (No login)

| URL | Description |
|---|---|
| **[pad.riseup.net](https://pad.riseup.net/)** | Etherpad-powered, privacy-focused — Riseup stores no IP addresses, requires HTTPS, and pads auto-delete after 60 days of inactivity. Generates a random URL on creation. |
| **[etherpad.wikimedia.org](https://etherpad.wikimedia.org/)** | Public Etherpad instance by Wikimedia. Anyone can create a new "pad", each with its own URL — anyone who knows the URL can edit it. |
| **[cryptpad.fr](https://cryptpad.fr/)** | End-to-end encrypted platform with real-time collaborative editors. No registration required, and content is encrypted on your device before being sent. Generates a unique random URL per document. |
| **[demo.hedgedoc.org](https://demo.hedgedoc.org/)** | HedgeDoc is an open-source, web-based collaborative Markdown editor. When you create a new note, it is given a long random ID used for accessing the editor. |
| **[hackmd.io](https://hackmd.io/)** | Polished hosted version of HedgeDoc — rich Markdown editor, generates a random URL per note, free tier with no login needed to view/edit shared notes. |

### 🔑 Key difference from the Tiptap demo

The Tiptap template at `template.tiptap.dev` is specifically a **Notion-style block editor** (slash commands, drag handles, etc.) with real-time collab. The closest equivalent in terms of richness + instant URL is **HackMD** or **CryptPad's rich text mode**. If you want plain-text speed, **pad.riseup.net** or **Etherpad** instances are the fastest with zero friction.
